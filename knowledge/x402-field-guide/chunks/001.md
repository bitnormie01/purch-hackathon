## Quickstart — First Vault Call in 5 Minutes

**Install:**
```bash
npm install @x402/fetch @x402/svm @solana/kit bs58
```

**Setup + search (costs $0.01 USDC):**
```typescript
import { createKeyPairSignerFromBytes } from "@solana/kit";
import { wrapFetchWithPayment, x402Client } from "@x402/fetch";
import { ExactSvmScheme, SOLANA_MAINNET_CAIP2 } from "@x402/svm";
import bs58 from "bs58";

// Decode your base58 secret key (64-byte Solana CLI keypair)
const secretKeyBytes = bs58.decode(process.env.WALLET_SECRET_KEY!);
const signer = await createKeyPairSignerFromBytes(secretKeyBytes);

const client = new x402Client();
client.register(SOLANA_MAINNET_CAIP2, new ExactSvmScheme(signer));
const fetchWithPay = wrapFetchWithPayment(fetch, client);

// Search — $0.01 USDC auto-paid
const res = await fetchWithPay("https://api.purch.xyz/x402/vault/search?limit=5");
const { items } = await res.json();
console.log(items); // Array of skills, knowledge packs, and personas
```

**That's it.** The payment is automatic. Section 3 has the full buy + download flow. Section 4 has every gotcha you'll hit.
